Portfolio Website

GitHub Repo

This portfolio website is constructed using standard development scripting in HTML, CSS, and basic JavaScript. It is hosted on AWS using the following services:

  • S3 for static hosting
  • CloudFront for content distribution
  • Route 53 for DNS management
  • CI/CD pipelines for continuous integration and deployment
  • IAM for orchestrating connections and managing permissions

As an API tech writer, this project demonstrates my ability to manage and document a wide range of skills, including web development and cloud infrastructure. The use of IAM ensures secure and efficient management of the workflow for updating the S3 bucket and other necessary procedures.

Check out the live site and explore the code on GitHub!
